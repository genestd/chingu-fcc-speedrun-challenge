FRED

This is a simple express server I built to access the FRED API at api.stlouisfed.org

I needed access to the data to build a D3 demo, but the API does not support cross-origin browser requests. I built this simple server to retrieve datasets in JSON format.

To use this proxy, submit the form below -OR- make a GET request to https://accidental-snowman.gomix.me/api/FRED? and then pass the full URL for the FRED query.
The server does no validation and only handles file_type=JSON - for example:
https://api.stlouisfed.org/fred/series?series_id=GNPCA&api_key=abcdefgh1234&file_type=JSON
