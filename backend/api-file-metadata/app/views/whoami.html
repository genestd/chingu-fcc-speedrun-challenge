HTTP Headers

This API was created to complete the FreeCodeCamp HTTP Headers Microservice Challenge. It will return a JSON object with the format:

{"ipaddress":"1.0.0.1" ,"language":"en-US,en;q=0.8" ,"software":"Mozilla/5.0 (Macintosh; Intel Mac OS X 10_12_0) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/56.0.2924.87 Safari/537.36" }

Options:

  • Endpoint: /api/whoami/
  • Parameters: None
