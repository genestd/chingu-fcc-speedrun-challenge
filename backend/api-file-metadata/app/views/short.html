URL Shortener

This API was created to complete the FreeCodeCamp URL Shortener Microservice Challenge. Given a URL for a valid domain in the format [protocol]:[domain]/[path], it will return a JSON object in the format:

{"original_url":"www.google.com","short_url":1}

Options:

  • Endpoint: /api/url/[:url]
  • [:url]: A valid URL in the format [protocol]:[domain]/[path]
  • Methods: Supports GET and POST methods
