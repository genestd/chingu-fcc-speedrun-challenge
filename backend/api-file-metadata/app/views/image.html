Image Search Abstraction

This API was created to complete the FreeCodeCamp Image Search Abstraction Layer Microservice Challenge. It uses the Pixabay image search API to return image information. It will return an array of JSON objects with the format:

{"previewHeight":112 ,"likes":1 ,"favorites":1 ,"tags":"plumbing, fittings, pipe" ,"webformatHeight":480 ,"views":1214 ,"webformatWidth":640 ,"previewWidth":150 ,"comments":0 ,"downloads":425 ,"pageURL":"https://pixabay.com/en/plumbing-fittings-pipe-connection-1002142/" ,"previewURL":"https://cdn.pixabay.com/photo/2015/10/22/23/08/plumbing-1002142_150.jpg" ,"webformatURL":"https://pixabay.com/get/e835b10d29f0033ed95c4518b7494795e470e2dd04b0154897f5c27ba1e8b5_640.jpg" ,"imageWidth":1920 ,"user_id":1552183 ,"user":"EugenesDIYDen" ,"type":"photo" ,"id":1002142 ,"userImageURL":"https://cdn.pixabay.com/user/2015/10/22/22-33-45-404_250x250.jpg" ,"imageHeight":1440}

Options:

  • Endpoint: /api/image
  • Methods: GET
  • Query String: search=SEARCH_TERM&offset=10
  • search (required): The string to search image tags
  • offset: starting page number of records to return - defaults to 1. The request will return 20 image details per request
  • Endpoint: /api/image/recent
  • Methods: GET
  • Returns: The 10 most recent search requests submitted through ../api/image
