Welcome

This site began as an anonymous proxy for the FRED API. Over time, it seemed a logical place to add other APIs as projects have required them.

Visit any of the landing pages for specific instructions on the APIs...or go straight to the endpoint if you are familiar wtih the usage.

# Landing Page API Endpoint Description
1 FRED /api/FRED/ Proxy for the FRED API at St. Louis Fed
2 Timestamp /api/timestamp/ FreeCodeCamp Timestamp API challenge
3 URL Shortener /api/url/ FreeCodeCamp URL Shortener challenge
4 File Metadata /api/metadata/ FreeCodeCamp File Metadata challenge
5 Exercise Tracker /api/exercise/ FreeCodeCampExercise Tracker challenge
6 Who Am I? /api/whoami/ FreeCodeCampHTTP Headers challenge
7 Image Search Abstraction /api/image FreeCodeCampImage Search Abstraction challenge
