Timestamp

This API was created to complete the FreeCodeCamp Timestamp Microservice Challenge. It will return a JSON object with the format:

{"unix":1488466199630,"utc":"Thu, 02 Mar 2017 14:49:59 GMT"}

Options:

  • Endpoint: /api/timestamp/[:time]
  • [:time]: A date string compliant with ISO-8601 or Unix timestamp
  • No parameter: Returns the current time
